This short video demo uses Xilinx's ZCU102 board with Zynq FPGA device
to show network and internet connectivity using the the LeWiz's LMAC1 core.

The LMAC1 core is implemented with Xilinx's DMA core.
OS - Petalinux

Demonstrated operations:
Local Network - PING, TELNET

Remote Internet operation - WGET large file from remote internet server
